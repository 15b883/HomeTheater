### 4K

**4K蓝光原盘和4K Remux**

原盘：正片视频+花絮+所有音轨

remux：正片视频+主要音轨

[个人家庭影院玩法](./home-video.md)

### 字幕

ass、ssa扩展名的是特效字幕，带有颜色、大小等特效

srt字幕无特效，均可用记事本打开、编辑。

加载字幕的方法，在播放视频时，将字幕文件直接“拖”到播放窗口
